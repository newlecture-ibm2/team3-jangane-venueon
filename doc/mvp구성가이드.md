# 프로젝트 기능 명세서 및 권한 구조 (Event Platform)

## 1. 역할 정의 (Roles)
- **Admin**: 시스템 전체 관리 및 모든 데이터 제어 권한
- **Host**: 이벤트 기획 및 운영 주체 (이벤트 등록/수정)
- **User**: 서비스 이용자 (이벤트 참여, 결제, 커뮤니티 활동)

---

## 2. 기능별 상세 권한 (ACL)

### [이벤트 관리]
- **이벤트 CRUD**: Host (본인 것만), Admin (전체)
- **이벤트 상세 수정**: Host (본인 것만)
- **이벤트 리스트/상세 조회**: All (User, Host, Admin)

### [티켓팅 및 결제]
- **티켓 예매 및 더미 결제**: User
- **구매 완료 페이지 확인**: User (본인 예매 건)

### [커뮤니티]
- **커뮤니티 글 CRUD**: All (본인이 작성한 글에 한함)
- **이벤트별 커뮤니티 CRUD**: All (본인이 작성한 글에 한함)
- **상세 내용 조회**: All

### [마이페이지]
- **활동 내역 관리**: User, Host (자신이 생성한 CRUD 내역 확인 및 수정)
- **구매/예약 내역 확인**: User
