1. 프로젝트 개요

"누구나 호스트가 되는 세상"

VenueOn은 공연, 강연, 워크숍 등 다양한 이벤트를 쉽게 기획하고 운영하며, 참가자들과 긴밀하게 소통할 수 있는 올인원 플랫폼입니다.

  • 이벤트 생명주기(기획~종료) 자동화
  • 풍부한 컨텐츠 표현력 (Rich Editor)
  • 정교한 지역 기반 추천 시스템

3. 핵심 기능

🛠 호스트 센터

이벤트 등록, 티켓 설정, 세션(온/오프라인) 관리 및 모집 상태 제어

💳 결제 및 티켓팅

Toss Payments 연동. 실시간 좌석/인원 관리 및 간편 결제

👥 특화 커뮤니티

단순 게시판을 넘어, 이벤트 참가자 인증 기반의 독점 정보 공유 채널 및 네트워킹 공간

🔒 보안 링크

구매자에게만 공개되는 전용 온라인 입장 링크 보안 시스템

4. 기술 스택

Frontend

  • Next.js 14 (App Router)
  • TypeScript
  • Tiptap (Rich Editor)
  • Vanilla CSS / CSS Modules

Backend

  • Java 21 / Spring Boot 3.x
  • PostgreSQL 15
  • Gradle
  • Docker
API & Others: Toss Payments, Kakao Postcode API, Git, GitHub Actions

5. 시스템 아키텍처

Hexagonal Architecture

  • Core Domain: 비즈니스 규칙의 독립성 확보
  • Adapters: 인프라 기술(DB, Web) 변경에 유연한 대응
  • Model: Event - Session - Ticket 계층 구조

6. 기술적 과제 (1)

세션 중심의 유연한 모델 설계

Challenge: 같은 이벤트라도 다양한 장소와 시간이 혼재된 복잡한 구조

Solution: v6 아키텍처 도입. Event - Session - Ticket으로 계층화하여 온/오프라인이 혼합된 하이브리드 이벤트 대응

6. 기술적 과제 (2)

컨텐츠 가독성 및 보안 강화

Tiptap Editor

단순 텍스트를 넘어 이미지/포맷팅이 포함된 풍부한 설명 페이지 지원

Address Filtering

Kakao API 기반 시/도-시/군/구 2단계 필터로 정확한 지역 탐색
