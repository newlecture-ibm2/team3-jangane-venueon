| 🔵 카테고리 선택 | 🔵 추가 요청 | 어드민이 관리하는 카테고리 목록에서 선택 |

| 🔵 온라인 세션 링크 | 🔵 추가 요청 | 온라인 선택 시 Zoom 등 세션 URL 입력 |

#### 취소 확인 모달

> 변경 사항이 **있을 때만** 모달 표시. 변경 사항이 없으면 바로 대시보드로 이동.

| 요소 | 내용 |
|------|------|
| **제목** | 강의 작성을 중단하시겠습니까? |
| **본문** | 지금 나가면 입력한 정보가 저장되지 않고 모두 삭제됩니다. 작성을 취소하시겠습니까? |
| `[계속 작성하기]` 버튼 | 모달을 닫고 작성 화면 유지 |
| `[나가기]` 버튼 | 작성 중인 데이터를 파기하고 `/host/dashboard`로 이동 |

| 기능 | 구분 | 설명 |
|------|------|------|
| 🟠 변경 사항 감지 (dirty check) | 🟠 필수 | 폼 데이터 변경 여부 추적하여 취소 모달 표시 조건 판단 |
| 🟠 브라우저 뒤로가기 가드 | 🟠 필수 | 변경 사항 있을 때 브라우저 뒤로가기/새로고침 시에도 경고 |

## 📌 우선순위 제안

### Phase 1 — 핵심 인프라 + 인증
1. 이메일 회원가입/로그인 + 이메일 검증 + 임시 비밀번호
2. Google OAuth2 소셜 로그인
3. 토스 페이먼츠 테스트 모드 연동
4. 알림 시스템 인프라 (알림 테이블 + API + 헤더 알림 아이콘)

### Phase 2 — 핵심 고도화 (MVP 확장)
5. 이벤트 에디터 폼 + 리뷰 시스템
6. 이벤트 지역별/캘린더 날짜별 필터링 + 페이지네이션
7. 마이페이지 고도화 (내 강의 탭 필터, 커뮤니티 4탭 통합, 알림 센터)
8. 관심 카테고리 등록 → 메인 홈 관심 이벤트 노출
9. 찜 목록 + 수강 바구니
10. 결제 완료 페이지 + 호스트 결제 내역/환불 관리

### Phase 3 — 커뮤니티 + 신고 체계
11. 커뮤니티 고도화 (좋아요, 대댓글, 공지, 게시글 상단 고정, 인기글)
12. 게시물 저장(북마크) 기능
13. 어드민 고도화 (회원 정지/권한, 카테고리, 관심 카테고리 관리)
14. 신고 시스템 고도화 (처리 단계, 제재 상태, 이의 제기)
15. 요청 게시판 (호스트 → 어드민/커뮤니티 관리자)

### Phase 4 — 뱃지 + 매칭 시스템
16. 뱃지 시스템 (자동 발급, 노출 설정)
17. 뱃지 보유자 검색/초대
18. 뱃지 기반 커뮤니티 개설 + 어드민 승인
19. 전체 통합 공지 게시판
