Section 02

타겟 사용자 & 정책

🏢 기획자 (HOST)

  • 기업 · 공공기관 · 사업자
  • 이벤트 생성 · 관리 · 티켓 판매
  • 사업자등록번호 / 기관 인증 가입
  • 기업/기관 전용 프로필 페이지 보유

👤 일반 사용자 (USER)

  • 개인 사용자
  • 이벤트 탐색 · 티켓 구매 · 커뮤니티 참여
  • 이메일 기반 일반 가입
  • 구매내역 · 참여 이력 관리
항목정책
이벤트 생성HOST만 이벤트 생성 가능 (사업자/기관 인증 필요)
이벤트 참여USER 이벤트 탐색 · 티켓 구매 · 참여
작성자 검증이벤트·커뮤니티·게시글 — 본인 작성 콘텐츠만 수정/삭제
향후 확장관리자(ADMIN) 역할 추가 가능 구조 유지

Section 05

기술 스택

⚛️ Frontend

  • Next.js 14+ (App Router, SSR/SSG)
  • Vanilla CSS Module (컴포넌트별 스코프)
  • React 18

☕ Backend

  • Spring Boot 3.x, Java 17
  • Hexagonal Architecture (Ports & Adapters)
  • Spring Cloud Gateway + OpenFeign

🗄️ Data & Cache

  • PostgreSQL 15 (서비스별 독립 DB ×3)
  • Redis 7 (JWT 블랙리스트, 캐시)
  • Spring Security + JWT (Access+Refresh)

🛠️ Infra & DevOps

  • Docker + Docker Compose
  • GitHub Actions (CI/CD)
  • 포트원(PortOne) 결제 · Swagger 문서
